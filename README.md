# ⏱️ Momentick

> 계획하고, 집중하고, 성장하세요.

Momentick은 일상의 시간을 더 의미있게 만들어주는 모바일 타임트래커 앱입니다.
할 일을 계획하고 시간을 기록하며 나만의 루틴을 만들어보세요.

---

## 📱 주요 기능

- ⏰ **타임 트래커** - 작업별 시간 측정 및 기록
- 📅 **일정 관리** - 할 일 계획 및 스케줄링
- 📊 **통계** - 시간 사용 패턴 분석
- 🔔 **알림** - 작업 시작/종료 리마인더

---

## 🛠 기술 스택

- **Framework**: React Native
- **Platform**: iOS / Android
